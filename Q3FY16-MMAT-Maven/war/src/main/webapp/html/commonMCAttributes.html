__belowHinavContent optional below navigation area Identifies content to be displayed directly beneath the hinav component __gaSearchDomain yes navigation area the domain name for searching within the current application scope. This value should be enclosed within option open and close tags, with selected='true' within the open option tag __hinavContent yes navigation area Identifies hinav content available within your web application. For JSP developers, the value may be either
  • a reference to an html file containing a static hinav, such as /WEB-INF/pages/contents/staticHinav.html
  • or in-line html hinav content
For Velocity developers, the value of this attribute should refer to a static .html or .vm file __hinavURI yes navigation area Used together with __hinavXmlPath to dynamically retrieve hinav content from CEC. This value represents the uri parameter used in the hinav generation script on CEC. This attribute will not be used if __hinavContent is defined. __hinavXmlPath yes navigation area Used together with __hinavURI as an additional attribute to dynamically retrieve hinav content from CEC. This value represents the xmlpath parameter used in the hinav generation script on CEC. This attribute will not be used if either
  • __hinavContent is defined (presume local content overrides remote content)
  • or __hinavURI is not defined as both URI and xmlPath are required for hinav retrieval from CEC
If this value is not provided, it will be set to the value of __hinavURI.
